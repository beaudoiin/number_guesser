# New Menu System In Development
This should be a re-usable menu that can be customized and implemented by the user in multiple apps.

# Features
* ansi, and ConsoleColor
* Adjusting based on window size (within limits)
* Layout Type, Single Column, Double Column, Double Row
* Menu item class which you add to a list
* Menu Item grouping with Foreground and Background color choices but ansi and ConsoleColor
* Specify the Colors of the Menu and the groups
* Transition from a background
* Optional Future, update screen by reading buffer and keeping track of where text is, reducing flicker.
* Default selector position when starting the menu
* Add a title
* Add a seperator for the title, specify character and padding and colors
* Adds description for menu items where user specified colors and border character for Top, right, left, bottom.
* Allows music to be played by passing the Play function of the instance of the sound. Not handleded within the menu system, only accepts function, therefore you coul dimplement other features.
* Specify custom navigation ConsoleKeys
* Specify Screen Caption / instructions for the menu
